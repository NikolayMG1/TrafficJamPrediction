# TrafficJamPrediction

Setup:
1. Python 3.11 or 3.12 is recommended
2. Install JDK (version 17 is recommended)
3. Setup kafka:
  - Demo: https://manishankarjaiswal.medium.com/installing-and-running-apache-kafka-on-windows-a-step-by-step-guide-part-1-7e957942e841
  - pip install requests kafka-python
4. Setup pyspark:
  - Demo: https://medium.com/@deepaksrawat1906/a-step-by-step-guide-to-installing-pyspark-on-windows-3589f0139a30
  - pip install pyspark==3.4.3 (any version before 4.1 should be ok)
